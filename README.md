# mental-health-prediction-system


Tech Stack: Python, LightGBM, Scikit-learn, Pandas 
Duration: June 2025 
Type: Internship Project - National Institute of Technology, Trichy

Overview
Developed a machine learning-based mental health prediction system using wearable sensor data to identify potential mental well-being risks and support early intervention.

Key Features
LightGBM-based predictive modeling
K-fold cross-validation for robust evaluation
Real-time risk assessment and classification
Data preprocessing and feature engineering
Analysis of wearable sensor data from 1,078 participants

Technologies Used
Python, LightGBM, Scikit-learn, Pandas, NumPy
